# Диаграммы модели C4
C1 — Контекстная диаграмма (System Context): показывает систему в окружении пользователей и внешних систем. 

C2 — Диаграмма контейнеров (Containers): высокоуровневое разбиение системы на исполняемые компоненты (мобильное приложение, база данных, веб-сервер и т.д.).

C3 — Диаграмма компонентов (Components): внутренняя структура каждого контейнера, основные модули и их взаимодействие.

C4 — Диаграммы кода (Code): детали реализации (классы, интерфейсы), обычно опускается или используется только для самых важных частей.

# MVP системы контроля проектной команды

## **1) MVP пользовательского интерфейса**

**Основные экраны и функциональность:**
- **Экран авторизации:**
  - Форма входа с полями email/пароль
  - Кнопка регистрации
  - Восстановление пароля
- **Дашборд пользователя:**
  - Список текущих задач
  - Индикаторы прогресса проектов
  - Уведомления
  - Профиль пользователя
- **Экран проекта:**
  - Информация о проекте
  - Список задач
  - Участники проекта
  - Документы
- **Экран задачи:**
  - Детализация задачи
  - Комментарии
  - Прикрепленные файлы
  - Таймлайн изменений
- **Профиль пользователя:**
  - Личная информация
  - Статистика активности
  - Настройки уведомлений

**Приоритетные элементы интерфейса:**
1) Интуитивно понятная навигация
2) Визуальные индикаторы статусов
3) Система уведомлений
4) Быстрый доступ к важным функциям

## **2. MVP серверной части**

**Микросервисная архитектура:**
- **Auth Service:**
  - Аутентификация пользователей
  - Управление ролями
  - Хранение профилей
- **Project Service:**
  - Создание и управление проектами
  - Распределение задач
  - Контроль сроков
- **Task Service:**
  - Управление задачами
  - История изменений
  - Комментарии
- **Analytics Service:**
  - Сбор метрик
  - Формирование отчетов
  - Статистика эффективности

**Базовые компоненты:**
- **База данных:**
  - PostgreSQL для основного хранилища
  - Redis для кэширования
- **API Gateway:**
  - Маршрутизация запросов
  - Авторизация
  - Мониторинг
- **Очередь сообщений:**
  - Обработка асинхронных задач
  - Уведомления

**Интеграционные сервисы:**
- Система аутентификации
- Хранилище файлов
- Почтовый сервис
